# pure-html-form-cv
Documento HTML puro, sem CSS e JavaScript, que simula em um formulário as informações gerais solicitadas em uma ficha de emprego.

## Objetivo
Demonstrar o uso de tags de formulário HTML, principalmente:
- `<form>`
- `<label>`
- `<input>`
- `<select>`

## Tecnologias
- HTML
  
## Observações
Projeto criado para fins de estudo.
